ECON 549 Write a program in R that allows one to calculate the monthly mortgage payment given a certain interest rate, loan amount (i.e., mortgage) and amortization period. Assume the interest rate is compounded semi-annually. Using the program, calculate the monthly pension one would get if she had $1 million in a retirement fund at age 65, plans to live 25 years, and assumes a real rate of interest of 2.5% compounded semi-annually.

Set up model

The real rate of interest of 2.5% is compounded semi-annually, so the rate of interest we use to calculate the interest is 1.25% semi-annually. We can set the time interval is half year, then every time period, we draw 6 * monthly pension and get the interest from the average principle during the time period
$$\frac{1}{2} * ( p_{t} + (p_{t} - 6*m ))$$
.

The first difference equation to model this process will be:


$$ p_{t+1} = \frac{1}{2} * ( p_{t} + (p_{t} - 6*m )) * (1 + 0.0125) $$

where p is the principle, m is the monthly pension.

We can use the numerical method to get the solution for the equation since we have

# principle is 1 million
p = 1000000
# initial monthly pension
m = 8890
# a vector to hold the number for plot 
#principle <- {}
#principle <- c()
principle <- vector()
while(p>0){
        # reset principle back to 1 million for 
        p = 1000000
        for(i in 1:(2*25)){
                principle[i]= p
                p = (( p+ (p - 6*m))/2) * (1 + 0.0125)  
                
        }
m = m+1

}
